# Wonderful Edge Tabs

**Wonderful Edge Tabs** 是一款为 VS Code 打造的垂直标签页管理工具。它的设计灵感来源于 Microsoft Edge 浏览器的垂直标签页合并功能，旨在解决打开过多标签页时顶部栏拥挤、难以查找文件的问题。

## ✨ 功能特性

- 垂直列表展示：在侧边栏以清晰的树状结构查看所有打开的标签。
- 自定义分组：通过右键菜单将相关的标签页合并到自定义分组（如“前端”、“后端”、“待办”）。
- 自动归类：未手动分组的文件会自动进入“未分类”区域。
- 状态同步：实时追踪当前激活的文件，并提供视觉反馈。
- 数据持久化：即使重启 VS Code，你的手动分组信息也会被保留。

## 🚀 使用方法

1. 安装插件后，点击左侧活动栏的 **层级图标 (Layers)**。
2. 在“未分类”列表中找到你想归类的文件。
3. **右键点击**文件名，选择 **“移动到 Edge 分组”**。
4. 输入组名并回车，文件即被成功合并。
5. 点击文件旁边的 **“X”按钮** 可将其移回未分类。

## 🛠️ 配置与开发

如果你想自行编译此插件：

1. 克隆仓库。
2. 运行 `npm install`。
3. 按 `F5` 启动调试窗口。
